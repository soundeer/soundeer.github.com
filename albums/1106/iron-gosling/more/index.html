    Reviews for Iron Gosling

    by Tommy Ill


     Back to album

    Hi! This is another little EP/Mixtape thing I made, it's basically just the weird stuff that I've been making with samples that we could never clear... Laptop microphone, high production values etc. Yay!

    If you like these songs you should totally go check out all the artists I sampled because they are all pretty great.

    Also if you like this EP you should check out our new album New Hat and a Haircut if you haven't already!

    xoxo
